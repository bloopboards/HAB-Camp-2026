I2C (Inter-Integrated Circuit protocol) is a 2 way communication protocol. Devices connected via I2C must operate at the same voltage. If they do not, a logic level converter will be necessary.

Basic Wiring for I2C communication

Gnd >> Gnd (Ground)
V   >> V (Supply Voltage)
SDA >> SDA (Serial Data)
SCL/SCK >> SCL/SCK (Serial Clock)


Required Library

The Wire library will be necessary in order to communicate with I2C devices. If you download a library for an I2C device without the Wire library installed the IDE will prompt you to download it.

The wire.begin(); command enables I2C communication. This is the only command you will need to use the in library, the rest will be done through each individual sensor's library.
