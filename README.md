# HAB-Camp-2026
WA Pierce College @ Fort Steilacoom's High Altitude Balloon Camp.

Every summer since 2024, Pierce College's Science Dome has been hosting a HAB Camp for highschoolers to attend and learn about the engineering process hands-on.
Summer interns usually have a month long preparation phase, where we design experiments, create our own payload, and write sensor code. 
Then, we host a 8-day camp over 2 weeks, launching multiple payloads into the stratosphere and collecting data to present at UW's research synopsium. 

Starting summer of 2026, this is where we share what code we've adapted, both by intern and camper.
Old code can be found in the 2025 HAB Folder.
