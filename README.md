# HAB-Camp-2026
WA Pierce College @ Fort Steilacoom's High Altitude Balloon Camp.
Every summer since 2024, Pierce College's Science Dome has been hosting a HAB Camp for highschoolers to attend and learn about the engineering process hands-on.
Summer interns usually have a month long preparation phase, where we design experiments, our own payloads, and write code. Then, we host a 8-day camp over 2 weeks. Starting summer of 2026, this is where we share what code we create, both by intern and camper. 

Collaborator Plans:
Currently there is a 2025 HAB folder with code that we use as building blocks (and cheatcodes). We could refactor/simplify them to be more general for campers to build upon, in which case those payloads with different forms of the same "template" code will be uploaded (hopefully) in specific payload folders.
