# HAB-Camp-2026
WA Pierce College @ Fort Steilacoom's High Altitude Balloon Camp.
Every summer since 2024, Pierce College has been hosting a HAB Camp for highschoolers to attend and learn about the engineering experience hands-on.
Summer interns usually have a month long testing phase, where we design experiments, our own payloads, and write code. Starting summer of 2026, this is where we update and maintain all code, (mostly) organized per specific payload made by campers and interns.
