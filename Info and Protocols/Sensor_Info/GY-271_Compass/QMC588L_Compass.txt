Overview
--------
The GY-271 is the breakout board for the QMC588L_Compass. The QMC588L is a 3 axis magnetic field sensor. This means that it can measure the magnitude of the magnetic field around it in 3 different directions (X,Y,Z). It contains internal circuitry which can convert the magnetic field strength on each axis into a heading.

Interface
---------
The GY-271 communicates via the I2C protocol and operates at 5V.

Library
-------
