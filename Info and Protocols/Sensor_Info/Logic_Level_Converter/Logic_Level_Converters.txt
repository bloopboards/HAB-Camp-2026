In order to facilitate communication between 2 devices which operate at different voltages a logic level converter will be necessary. 

The Arduino boards you have been equipped with operate at 5 volts. Some of our sensors operate at 3.3 volts. this is a problem because if you connect a 3.3 sensor with a 5V board the Arduino will not be able to receive information from the sensor and the sensor may be damaged. this is where the logic level converter comes in.

The logic level converter functions like a translator, taking 5v pulses from the board and converting them to 3.3V signals for the sensor, and vise versa.

Use the following setup when wiring a logic level converter.

Refer to LL Converter ref.png
 
** NOTE **
The logic level converter has a HIGH voltage side and a LOW voltage side. do not get them mixed up


ARDUINO		LL CONVERTER		SENSOR

3.3V	>>	LV		>>	Sensor
5V	>>	HV		
GND	>>	GND		>>	GND
SDA	>>	Chanel A	>>	SDA
SCL	>>	Chanel B	>>	SCL
		GND		<<	INT
