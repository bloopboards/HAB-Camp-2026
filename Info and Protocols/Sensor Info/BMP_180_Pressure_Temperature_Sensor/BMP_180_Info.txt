Overview
--------
The BMP_180 is a pressure and temperature sensor. It includes interal circuitry that will also calculate the altitude using pressure data.

Interface
---------
Uses I2C communication protcol. Operates at 5V.

Library
-------
Use the Adafruit BMP085 Library. This is not a typo, the BMP085 library is used for both the BMP085 and BMP180. If prompted to download any dependencies do so.
