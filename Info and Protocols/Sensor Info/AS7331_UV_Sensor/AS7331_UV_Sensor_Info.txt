Overview
--------
The AS7331 is a spectral UV sensor. It can detect 3 different wavelength bands of UV light: 
UV A (315–400 nm)
UV B (280–315 nm)
UV C (100–280 nm)
For reference, wavelength visible light is 380-750 nm.

Interface
---------
The AS7331 uses the I2C communication protocol. The sensor operates at 3.3 volts so you must use a logic level converter.

***WARNING***
Failure to use a logic level converter may damage the sensor.
*************

Library
-------
Install the SparkFun AS7331 library. The SparkFun AS7331 library is dependent on the Wire library. If prompted to download any other dependencies do so.
