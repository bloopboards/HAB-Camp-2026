BMP 180 Library Install Guide

INSTALL THE ADAFRUIT BMP LIBRARY IN ARDUINO IDE

1) Open Arduino IDE

2) Top Left toolbar, go to "Tools" -> "Manage Libraries..." Wait for indexing.

3) Type into the top right search bar, "Adafruit BMP"

4) Choose "Adafruit BMP085 Library"

5) Select the latest version and click "Install"

6) A window asking to download dependencies may pop up

7) Click "Install all"

8) The library should show "INSTALLED" in the library manager.

9) Done
