DS18B20 Library Install Guide

INSTALL THE DallasTemperature Library IN ARDUINO IDE

1) Open Arduino IDE

2) Top Left toolbar, go to "Tools" -> "Manage Libraries..." Wait for indexing.

3) Type into the top right search bar, "Dallas Temperature"

4) Choose "Dallas Temperature by Miles Burton"

5) Select the latest version and click "Install"

6) A window asking to download dependencies may pop up

7) Click "Install all"

8) The library should show "INSTALLED" in the library manager.

9) Done
